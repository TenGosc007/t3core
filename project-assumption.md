# Tic Tac Toe

A core for the game tic tac toe

## Assumptions

- 3x3 board
- game mode - PVP
- user is able to input value
- there are 2 players
- program ends when there is a win or draw

## Steps

1. Game starts
2. 'O' starts
3. player picks the field from 1 to 9
4. next player 'X' selects the field
5. players repeat their choice every turn
6. If win or draw - show message
7. Program ends
